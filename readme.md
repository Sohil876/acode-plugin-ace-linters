# Ace Linters

**Ace Linters** is a powerful plugin for [Acode Editor](https://github.com/Acode-Foundation/Acode) that integrates the [ace-linters](https://github.com/mkslanc/ace-linters) library for providing code linting and code formatting for supported languages.

## ðŸ›  Usage

### Linting

Linting is active automatically. Open any supported file, and syntax errors will be highlighted in the gutter (sidebar) or underlined within the editor. Tap the error icon to see the specific error message.

### Formatting

Ace Linters will be available for supported languages in formatter option, select it and click on format to format your file.
